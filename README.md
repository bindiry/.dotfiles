# .dotfiles

### Install

Install Xcode

```
xcode-select --install
# install homebrew
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
# install git
brew install git
# clone repository
cd
git clone git@github.com:bindiry/.dotfiles.git
cd .dotfiles && chmod +x install && ./install
```
